## ▶️ Jak uruchomić projekt

1. **Sklonuj repozytorium**
```bash
git clone https://github.com/twoje-konto/TennisPlan.git
cd TennisPlan
```

2. **Zainstaluj zależności**
```bash
npm install
```

3. **Uruchom aplikację lokalnie**
```bash
npm run dev
```

4. Otwórz w przeglądarce:
```
http://localhost:5173
```
